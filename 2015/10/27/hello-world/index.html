hexo向google提交sitemap

  • hexo
菜屌日妓

一、安装sitemap

npm install hexo-generator-sitemap –save

在博客根目录下找到_config.yml，添加代码：

sitemap:
path: sitemap.xml

生成静态页

hexo g

进入localhost:4000/sitemap.xml即可查看到效果。

二、向谷歌提交sitemap

1.登陆Webmaster Central。

2.点击“ADD A SITE”红色按钮，输入网址点击continue按钮。

3.进入验证页面，有两种方法：

第一种推荐方法（Recommended method）：下载HTML验证文件放入根目录下，然后hexo生成提交命令之后点击Verify红色按钮验证。

第二种替代方法（Alternate methods）：获取html meta，将< meta >放入themes>yilia>layout>_partial的head.ejs中，然后hexo生成提交命令之后点击验证。

4.进入谷歌站长工具提交sitemap，页面地址
