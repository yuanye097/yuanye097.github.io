javascript indexOf()方法学习小记

  • javascript
学习日妓

Array.prototype.indexOf()

定义：indexOf()方法返回指定元素在数组中找到的第一个索引值，否则返回-1。

indexOf()语法：

arr.indexOf(searchElement[, formIndex = 0])

参数：

searchElement:要查找的元素
formIndex:开始查找的位置，如果该索引值大于或等于数组长度，意味着不会在数组里查找，则返回-1。如果参数中提供的索引值是一个负值，则将其作为数组末尾的一个抵消，即-1表示从最后一个元素开始查找，-2表示从倒数第二个元素开始查找，以此类推。
PS：如果参数中提供的索引值是一个负值，仍然是从前向后查询数组。如果抵消后的索引值仍小于0，则整个数组都将会被查询。其默认值为0。

描述：
indexOf使用比较运算符（===）进行判断searchElement与数组中包含的元素之间的关系

接下来学习下字符串中的indexOf()

String.prototype.indexOf()

定义：indexOf()方法返回指定值在字符串对象中首次出现的位置。从fromIndex位置开始查找，如果不存在，则返回-1。

indexOf()语法：

str.indexOf(searchValue[, fromIndex])

参数：
searchValue:一个字符串表示被查找的值。
formIndex:表示调用该方法的字符串中开始查找的位置。可以是任意整数。默认值是0。如果fromIndex<0则查找整个字符串(如同传进了0)。如果fromindex>= str.length,则该方法返回-1，除非被查找的字符串是一个空字符串，此时返回str.length。

描述：
字符串中的字符被从左向右索引。首字符的索引(index)为0，字符串stringName的最后一个字符的索引是stringName.length - 1。
