2015-10-27

hexo向google提交sitemap

一、安装sitemap

npm install hexo-generator-sitemap –save

在博客根目录下找到_config.yml，添加代码：

sitemap:
path: sitemap.xml

生成静态页

hexo g

进入localhost:4000/sitemap.xml即可查看到效果。
