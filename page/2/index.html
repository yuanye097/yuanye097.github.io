offset/client/scroll学习小记

昨天写了篇无缝滚动的学习小记，涉及到了scroll和offset知识点，这里我把我的理解的一些基础写下来，望大神给予指点。

一、offsetWidth/clientWidth/scrollWidth

1.无滚动条内容无溢出，如图所示：


根据图中所示，可看出
BOX.offsetWidth = 440px;包含了width+padding+border
BOX.clientWidth = 420px;包含了width+padding
Box.scrollWidth = 420px;包含了width+padding

2.有滚动条内容无溢出，如图所示：


根据图中所示，可看出
BOX.offsetWidth = 440px;包含了width+padding+border
BOX.clientWidth = 403px;包含了width+padding-滚动条宽度
Box.scrollWidth = 403px;包含了width+padding-滚动条宽度

3.有滚动条内容溢出，如图所示：


BOX.offsetWidth = 440px;包含了width+padding+border
BOX.clientWidth = 403px;包含了width+padding-滚动条宽度
Box.scrollWidth = 内容的宽度-滚动条的宽度;(比如内容有10个“BOX”，展现5个BOX溢出了5个BOX，这段宽度就是这10个BOX的宽度+padding-滚动条宽度)

二、offsetLeft/clientLeft/scrollLeft

1.无滚动条内容无溢出，如图所示：


根据图中所示，可看出
BOX.offsetLeft = BOX外边框距离BODY的边界的距离。
BOX.clientLeft = 10px;(box蓝色width左边界距离box实际width左边界的距离，即border！)
Box.scrollLeft = 0;

2.有滚动条内容无溢出，如图所示：


根据图中所示，可看出
BOX.offsetLeft = BOX外边框距离BODY的边界的距离。
BOX.clientLeft = 10px;(box蓝色width左边界距离box实际width左边界的距离，即border！)
Box.scrollLeft = 0;

3.有滚动条内容溢出，如图所示：


BOX.offsetLeft = BOX外边框距离BODY的边界的距离。
BOX.clientLeft = 10px;(box蓝色width左边界距离box实际width左边界的距离，即border！)
Box.scrollLeft = 内容向左边移动了多少像素，即移动距离。
